# Decent Public School Website

A state-of-the-art, responsive web platform for **Decent Public School, Guwahati**, crafted with Next.js 16, React 19, and Tailwind CSS 4. This project represents the school's digital presence, offering a premium user experience with modern aesthetics, smooth animations, and comprehensive information access.

This repository hosts the official source code for the Decent Public School website. The platform is designed to provide parents, students, and prospective applicants with a seamless interface to explore the school's academic curriculum, infrastructure, and latest updates.

**Key Design Principles:**
- **Visual Excellence:** A "wow" factor with curated color palettes, elegant typography, and glassmorphism effects.
- **Responsiveness:** Fully fluid layouts optimized for mobile, tablet, and desktop devices.
- **Performance:** Optimized for speed and SEO best practices.
